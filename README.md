## рџ“– About

**PlacePrep AI** is a comprehensive placement preparation platform built to help students prepare for campus placements completely **free of cost**.

The platform combines coding practice, aptitude preparation, interview preparation, resume building, company-specific roadmaps, progress tracking, and AI-powered learning into one modern web application.

Unlike paid platforms, PlacePrep AI is designed to make placement preparation accessible to every student.

## Backend Setup

Navigate to backend

```bash
cd backend
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Install Packages

```bash
pip install -r requirements.txt
```

Run Flask

```bash
python app.py
```

Backend runs at

```
http://localhost:5000
```
